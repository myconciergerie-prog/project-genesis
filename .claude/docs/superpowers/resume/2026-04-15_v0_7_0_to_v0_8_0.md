Composes the five shipped skills into the 7-phase protocol from `memory/master.md`:

| Phase | Content | Skill(s) invoked |
|---|---|---|
| **Phase -1** | Dependencies pre-flight (3-mode ladder, multidevice baseline) | `phase-minus-one` |
| **Phase 0** | Project seed loading — read `config.txt` + mixed media from the input folder | built-in file reads, no skill |

| **Phase 3** | Git repo init + SSH identity + first bootstrap commit | manual paste-back pattern, per-project SSH identity from Layer 0 |
| **Phase 4** | Project-specific seeds (master vision, design specs, initial skill stubs) | built-in Write |
| **Phase 5.5** | Auth pre-flight (SSH keygen, fine-grained PAT, empty repo creation, 3-probe test) | `phase-5-5-auth-preflight` |
