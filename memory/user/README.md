# memory/user/

Project-scoped user notes. **Most user profile content lives in Layer 0** (`~/.claude/CLAUDE.md`) — this directory holds only the bits that are specific to working on Project Genesis and don't generalize to other projects on this machine.

## What Layer 0 already covers

- Identity (primary email, multi-account setup)
- Language (French native, English fluent, R9 policy)
- Working style (step-by-step validation, `autonome` / `termine tout` / `pause` keywords)
- Async mid-flow questions preferred pattern
- AI stack (Claude Max 20x, Anthropic API, Gemini, GPT)

Read `~/.claude/CLAUDE.md` User profile section for the full content. This Genesis-local directory does not duplicate it.

## What would go here

Only content that is **Genesis-specific AND user-facing**. Examples that would qualify:

- A note about the user's preferred style for Genesis skill design (if it ever diverges from the general working style)
- A specific Genesis quirk the user has about wording / naming that doesn't apply elsewhere

Nothing currently exists here. Entries will be added as the project grows and concrete user-facing patterns diverge from the universal profile.
