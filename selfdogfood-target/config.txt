vision: |
  A Claude Code plugin that transforms a folder containing a config.txt seed
  (plus optional mixed media — briefs, PDFs, images, transcripts, URLs) into
  a fully bootstrapped project: rules, memory architecture, research cache,
  git repo, and resume prompts. Orchestrated by Claude Code itself via a
  7-phase protocol. Minimizes user friction to the security floor of
  unavoidable manual interventions.

  v1 shipped. v1.2 is about hardening through self-dogfood before v2.
  v2 target: Promptor fusion — replace config.txt with a drag-and-drop box
  where the user drops any document and Claude extracts a structured brief,
  then formalizes through at-most-3 conversational questions. One-click auth.
  Boomer-friendly. "Dis-moi ton idée, je m'occupe du reste."

  Target user for v2: Victor, 77 years old, has a magnificent idea, no
  technical vocabulary, hears a soft chime when his project is born on
  GitHub.
