Tuto d'installation

Pas besoin d'être un pro. Suis les étapes dans l'ordre, et en 10 minutes c'est prêt.

1

Ouvre le « Terminal » de ton ordinateur

C'est une fenêtre où tu peux taper des commandes. Sur Windows 11, c'est tout simple :

📄   Applis et fonctionnalités
⚙   Paramètres
📁   Explorateur de fichiers
⌨   Terminal
💻   Gestionnaire de tâches

↑ Clic droit sur le bouton Démarrer (en bas à gauche) → clique sur Terminal

2

Installe Claude Code

Copie-colle cette ligne dans le Terminal et appuie sur Entrée :

Terminal
PS C:\Users\toi> irm https://claude.ai/install.ps1 | iex
Installing Claude Code...
✓ Claude Code installed successfully!

Une seule fois. Le script installe tout automatiquement.

3

Prépare ton dossier projet

Crée un dossier n'importe où (sur le Bureau par exemple) et mets-y tout ce qui décrit ton projet :

📄 Un fichier texte — ton idée en quelques lignes (même 3 suffisent)
📷 Des photos — ton produit, ton local, tes références
📎 Des PDFs — menu, tarifs, cahier des charges
🔗 Des liens — dans un fichier texte, un lien par ligne

Pas d'idée de quoi écrire ? Même un dossier vide fonctionne — Genesis te guidera.

4

Ouvre Claude Code dans ton dossier

Le plus simple sur Windows 11 : ouvre ton dossier dans l'Explorateur, puis :

📁   Ouvrir
✎   Renommer
⌨   Ouvrir dans le terminal
📋   Copier
🗑   Supprimer

↑ Clic droit dans le dossier (sur un espace vide) → Ouvrir dans le terminal

Ensuite, tape claude et appuie sur Entrée :

Terminal — mon-projet
PS C:\Users\toi\Desktop\mon-projet> claude

███ Claude Code
Welcome! How can I help you today?
> _
5

Installe le plugin Genesis

Dans Claude Code, tape cette ligne :

> /plugin install project-genesis@myconciergerie-prog/project-genesis

Une seule fois. Après, Genesis est installé pour tous tes projets.

6

Lance Genesis !

Toujours dans Claude Code, dis simplement :

Claude Code — mon-projet
> Bootstrap this project using genesis-protocol

⚡ Genesis v1.0.0 activated
Analysing folder contents...
Found: config.txt, menu.pdf, 3 photos
Parsing project intent...

📋 Parsed intent card
Project: La Table de Marie
Vision: Site vitrine + réservation
Mode: Semi-auto

Proceed? (yes / edit / abort) yes

💡 Astuce permissions : Claude va te demander des autorisations pendant le processus (pour créer des fichiers, lancer des commandes…). Tape a pour tout accepter d'un coup, ou y pour valider une par une.

3 vitesses — à toi de choisir

⚡

Automatique — ~5 min

L'IA fait tout. Pause uniquement pour mots de passe et confirmations de sécurité.

⚙

Semi-auto — ~15 min ⭐ Idéal 1er test

L'IA montre ce qu'elle fait à chaque étape. Tu valides ou tu corriges.

📚

Pas à pas — ~45 min

L'IA t'explique tout. Tu tapes, tu apprends. Pour comprendre le « sous le capot ».
